# Shopify Hydrogen Web SDK Integration Skill

## API Information
- **Type**: Frontend Web SDK / Headless Commerce
- **Product**: Hydrogen Integration (`insider-hydrogen-web-sdk`)
- **Technology**: React, Shopify Liquid (Checkout)

### Hydrogen / Custom Storefront Integration (Academy)
- For **Hydrogen** or **custom storefronts**, the standard Shopify App Embed cannot inject Insider Object (IO) and JS Tag (ins.js). These must be **manually implemented** (contact Insider One for implementation guidance).
- If you change **category mapping** (e.g. from Collections to Tags or Product Type) in Shopify integration, you must also update the **taxonomy** object in the Insider Object (IO) so that Insider One reads category values from the correct field. See [shopify-onboarding-wizard.md](shopify-onboarding-wizard.md) / Integrate Insider with Shopify Store.

## Core SDK Components

### Provider Setup
The application must be wrapped in `<Insider.Provider>`.
| Required Prop | Type | Description |
|---------------|------|-------------|
| `partnerName` | String | e.g., 'shopifytest' |
| `partnerId`   | String | Insider account ID |

*(Note: Web SDK and Insider Object cannot be enabled simultaneously in Gachapon to avoid data loss).*

### Tracking Methods
| Method | Description |
|--------|-------------|
| `Insider.pushHomePageView()` | Tracks homepage visits |
| `Insider.pushProduct(product)` | Tracks product views (needs id, name, price, etc.) |
| `Insider.pushCart(cart)` | Tracks cart state (needs total, items array) |

## UI/Integration Test Flows

### Flow 1: Verifying SDK Loading in Browser
```typescript
// Navigate to Hydrogen Storefront
await page.goto('https://shop.emreisik.dev/');

// Verify if the window.InsiderQueue is initialized by the SDK
const isSdkLoaded = await page.evaluate(() => {
return window.InsiderQueue !== undefined && Array.isArray(window.InsiderQueue);
});

expect(isSdkLoaded).toBeTruthy();
``` 

### Flow 2: Intercepting Hit Requests
```typescript
// Listen for tracking requests generated by the SDK
const hitPromise = page.waitForRequest(request =>
request.url().includes('hit.useinsider.com') && request.method() === 'POST'
);

// Trigger a product view (e.g., navigate to a product page)
await page.click('text="Yellow T-Shirt"');

const hitRequest = await hitPromise;
const postData = hitRequest.postDataJSON();

// Assert base64 decoded or raw payload structure
expect(postData.events[0].event_name).toBe('product_detail_page_view');
```

## Expected Behaviors & Checkout Limitations

- **Checkout Page Restriction**: Hydrogen delegates the actual checkout to Shopify Liquid. The React SDK **cannot** intervene on the checkout page. Multidomain helper and Shopify Web Pixels must be used instead.
- **Missing Parameters Error**: If `<Insider.Provider>` lacks `partnerName`, the SDK must gracefully throw a console error and prevent `ins.js` injection.
